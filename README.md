2. 关键词和搜索策略（包含同义词 & 多语言）

你可以先定义一套「模板」：

走路类模板：

{city} walking tour

{city} city walk 4K

{city} POV walk

{city} walking tour no commentary

{city} virtual walking tour

驾车/行驶类模板：

{city} driving tour

{city} 4K drive

{city} POV driving

{city} car drive through downtown

多语言增强（举例）：

中文：{城市名} 城市漫步、{城市名} 街景散步、{城市名} 开车巡游

日文：{city} 街歩き、{city} 散歩動画

其他语种：如西语 recorrido caminando {city} 等。

可以把这些模板 + 城市名自动笛卡尔积生成搜索 query 列表，再交给脚本/YouTube Data API 去跑。
